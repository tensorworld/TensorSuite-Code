# TensorSuite
This repository provides synthetic sparse tensor generation for TensorSuite: 
```text
https://tensorworld.github.io/TensorSuite/
```
Please visit the TensorSuite website for more information, including tensor format descriptions and real-world tensor datasets.

Repository-level `tensorsuiteIO/` contains the shared TensorSuite `.tns`,
`.tnsb`, metadata, README, and bundle validation helpers used by synthetic
generators such as `SynTensorGen/element-wise/random` and
`SynTensorGen/element-wise/power-law`.
