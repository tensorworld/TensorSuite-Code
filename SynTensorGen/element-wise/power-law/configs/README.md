# configs

Small YAML configurations for smoke tests and examples.

- `cp_fastskg_tiny.yaml`: builds a CPD seed and expands it with fastSKG.
- `example_cp_fastskg.yaml`: commented CP-fastSKG example for users to copy and edit.
- `fastskg_input_seed_tiny.yaml`: expands the tiny TensorSuite seed in `../seeds`.
- `zipf_tiny.yaml`: generates an independent Zipf baseline tensor.
