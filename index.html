Quick Start

Generate one tensor or a complete benchmark batch.

TensorSuite writes .tns coordinate-format tensors whose non-zero values are sampled from a standard normal distribution. The limited generator estimates memory use before writing data.

Single tensor python3 SynTensorGen/random_tensor_generator_limited.py output.tns D 2.0 50%1024 2%4096
Preset suite bash SynTensorGen/generate_all_tensors_limited.sh

Density patterns

Mix dense and sparse modes by passing one density%dimension argument per tensor mode.

Memory thresholding

Estimate NNZ and COO storage before generation, then skip workloads that exceed the configured GiB limit.

Benchmark batches

Generate combinations of 4D and 5D tensors with repeatable naming conventions for conversion and performance studies.

Output Format

Plain-text .tns files.

The first line stores the tensor order, the second line stores mode dimensions, and each following line stores one-based coordinates followed by a generated floating-point value.
